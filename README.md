## Como subir na Vercel

1. Crie um repositório no GitHub.
2. Envie TODOS os arquivos deste projeto para a RAIZ do repositório.
3. Na Vercel, clique em Add New > Project.
4. Escolha o repositório.
5. Framework Preset: Other.
6. Build Command: deixe vazio ou use `npm run build`.
7. Output Directory: deixe vazio.
8. Clique em Deploy.

## Importante

O arquivo `index.html` precisa ficar na primeira tela do repositório, não dentro de uma pasta extra.
